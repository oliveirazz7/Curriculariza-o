# Curriculariza-o
Curricularização da extensão parte 2

Com base nas perguntas que elaboramos e no nosso protótipo, iremos desenvolver uma interface simples, interativa e educacional para as crianças. Usaremos bastante imagens para facilitar o entendimento das questões. As questões foram elaboradas com base na história, geografia, cultura e conhecimentos gerais da cidade de Brusque.

Cada questão terá 3 alternativas, contendo apenas 1 correta. Ao finalizar as respostas, o site irá mostrar a nota do usuário.
